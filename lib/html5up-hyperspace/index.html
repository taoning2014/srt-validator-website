SRT Validator

Have an invalid srt file when upload to social media, and don't know how to fix it? We have solution for you:

An SRT validator library you can either add as npm dependency or try it online.

  • Learn how to use it

Step One

Upload a srt file to this website. If you don't have one, here is a sample file for you to play with.

After upload, its content will be shown at an editable textarea below the upload button.

  • Sample SRT

Step Two(optional)

Edit the content.

Step Three

Check whether the current file has valid srt format by click "Validate SRT" button. If it is invalid, there will be error message display below, you can come back to step two to modify it.

SRT is a quite simple, click the link below to see what is a valid format.

  • Learn more

Step four(coming soon)

Once the file is valid, you can download it from the download button.
